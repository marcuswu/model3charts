About model3data

model3data is a quick and dirty visualization of Tesla Model 3 invite, configuration, and VIN data. The amount of data presented here should grow over time. If you let me know what additional charts / data you would like to see here, I will work towards that. This site runs on a server I am paying for, so please scroll down and contribute if you feel like you are getting something out of it.

What

All of the data that is displayed here comes from the Tesla Model 3 invite spreadsheet which is an incredible data source kept up to date and managed by reddit user /u/Teslike. I refresh this site from that data periodically and cache it locally. This site is run on a combination of nginx, nodejs, MongoDB, and Polymer.

Why

The Tesla Model 3 invite spreadsheet already has all of this information. It is hosted by Google Sheets which has a user limit and can be fairly slow. I created this site to suppliment that spreadsheet and produce a (hopefully) faster experience.
